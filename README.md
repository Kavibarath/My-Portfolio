## Run the Portfolio

This is a static website, so the simplest way is to double-click `index.html`.

For a local development server, open the VS Code terminal and run:

```powershell
python -m http.server 5500
```

Then open this URL in your browser:

```text
http://localhost:5500
```

Edit these files:

- `index.html` for content and links
- `styles.css` for design
- `script.js` for the animated AI background
